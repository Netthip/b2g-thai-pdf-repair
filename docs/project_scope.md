# Project Scope — B2G Thai PDF Repair

## เอกสารเป้าหมาย
- ชื่อไฟล์: `เล่ม พรบ ประชาชน.pdf`
- ชื่อเรื่อง (เนื้อหา): งบประมาณฉบับประชาชน ประจำปีงบประมาณ พ.ศ. 2570
- ผู้จัดทำต้นฉบับ: สำนักงบประมาณ / สำนักนายกรัฐมนตรี (สร้างด้วย Canva)
- จำนวนหน้า: **72 หน้า** (รวมหน้าปก หน้าว่าง สารบัญ และหน้าท้ายเล่ม)

## เป้าหมาย
1. ตรวจจับ + บันทึกข้อผิดพลาดภาษาไทยทุกหน้า แบบ trace ได้
2. จัดทำแนวทางแก้ไข (แก้ที่ Canva) ที่อ้างอิง Issue ID
3. รักษาโครงสร้าง/เนื้อหา/ลำดับหน้า/ภาพ/ตาราง/เจตนาเดิมไว้ครบ
4. ตรวจซ้ำเป็นรอบจนผ่าน Quality Gate
5. ส่งมอบหลักฐาน QA + แพ็กเกจ GitHub

## นอกขอบเขต (Out of scope)
- การ rebuild เลย์เอาต์อินโฟกราฟิกใหม่ด้วยโค้ด (จะทำให้คุณภาพงานออกแบบตก)
- การแก้ตัวเลขเชิงนโยบาย/งบประมาณ (ตรวจความถูกของการ "พิมพ์" ไม่ใช่ตัดสินเชิงนโยบาย)
- การเผยแพร่ไฟล์ต้นฉบับสู่สาธารณะ

## แนวทางที่เลือก
**Audit + แก้ที่ Canva** (ผู้ใช้ยืนยัน) — ระบบออก Issue Registry แม่นยำ → แก้ใน Canva →
export ใหม่ → ระบบ render/QA ซ้ำจนผ่าน
