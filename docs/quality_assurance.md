# Quality Assurance & Quality Gate

## Quality Gate — จะถือว่า "ผ่าน" ต่อเมื่อ
- ข้อผิดพลาดภาษาไทยที่ยืนยันได้ = 0
- อักขระซ้อน / สระลอย / วรรณยุกต์ผิดตำแหน่ง = 0
- เว้นวรรคภายในคำที่ยืนยันได้ = 0
- ข้อความสำคัญ (ชื่อ เรื่อง วันที่ ตัวเลข หน่วยงาน หัวข้อ ตาราง) = ถูกต้อง
- หน้าไม่หาย ลำดับหน้าไม่เพี้ยน / ไม่มีข้อความล้น-ตัด-ทับ
- มีผล visual QA ครบทุกหน้า
- รายการที่ยืนยันไม่ได้ → ระบุเป็น "Human Review Required" พร้อมหน้าและเหตุผล

> **ห้ามปิดงานด้วยคำว่า "0 errors" หากยังมี Human Review Required ที่เกี่ยวกับข้อความ/ความหมาย**

## สถานะปัจจุบัน (รอบ 1) — ยังไม่ผ่าน Gate
| เกณฑ์ | ผล |
|---|---|
| ตรวจครบทุกหน้า (1–72) ระดับเต็มหน้า | ✅ |
| ข้อผิดพลาดที่ยืนยันแล้ว | ❌ พบ 1 (TH-001) — ยังไม่แก้ |
| ซูมยืนยันหัวเรื่องหมวดหลักทั้ง 5 | ✅ (พบเพี้ยน 1 = หน้า 66) |
| Human Review Required | ⚠️ 2 รายการ (รอบ 2 + ถ้อยคำ SME) |

→ **ยังไม่ผ่าน** เพราะมี 1 error ที่ยังไม่แก้ และยังต้องทำรอบ 2 (ซูมแถบหัวข้อทุกจุด)

## นิยามความรุนแรง
- **สูง**: หัวเรื่องหมวด/ชื่อเรื่อง/ตัวเลข/หน่วยงาน/วันที่ ผิด
- **กลาง**: เนื้อหา body ผิดจนเปลี่ยนความหมาย
- **ต่ำ**: ถ้อยคำกำกวม/รูปแบบ ไม่เปลี่ยนความหมายหลัก
