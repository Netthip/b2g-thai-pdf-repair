## TH-001 — หน้า PDF 66 (หน้าพิมพ์ 60)
**หัวเรื่องหมวด:** "ด้านการบริหารภาครัฐ การปฏิรูปกฎหมาย"

- อาการ: บรรทัดที่ 2 ของหัวเรื่องแสดง **"การปฏฐูปกฎหมาย"** (สระ "ิ" หาย, ร เพี้ยนเป็น ฐ)
- ต้องเป็น: **"การปฏิรูปกฎหมาย"**

**วิธีแก้:**
1. เปิดหน้าที่ 60 (หน้าหมวด "ด้านการบริหารภาครัฐ")
2. คลิกที่กล่องข้อความหัวเรื่อง บรรทัด "การปฏิรูปกฎหมาย"
3. **ลบทั้งคำ "ปฏิรูป" แล้วพิมพ์ใหม่** ว่า `ปฏิรูป` (อย่าแก้ทีละตัว เพราะอักขระเดิมเพี้ยน)
   - ตรวจให้เห็น: ป → ฏ → สระอิ (ิ) บน ฏ → ร → สระอู (ู) ใต้ ร → ป
4. **สำคัญ:** ตรวจว่าฟอนต์ของกล่องนี้รองรับภาษาไทยครบ
   - แนะนำใช้ฟอนต์ไทยที่มีกลิฟครบ เช่น **Sarabun / Noto Sans Thai / Kanit / IBM Plex Sans Thai**
   - หลีกเลี่ยงฟอนต์ Latin-only (เช่น Garet) กับข้อความไทย เพราะ fallback อาจทำคลัสเตอร์ "ปฏิ" แตก
5. เทียบกับหัวเรื่องหน้าอื่นที่ถูกต้อง (หน้า 68 / สารบัญ) ให้รูปคำตรงกัน
